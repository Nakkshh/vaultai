# VaultAI Frontend

Next.js frontend for VaultAI — semantic code search platform.

## Stack
- Next.js 16, TypeScript, Tailwind CSS
- Axios, js-cookie

## Local Setup

```bash
npm install
npm run dev
```

Runs on http://localhost:3005

## Environment Variables
NEXT_PUBLIC_API_URL=------

## Production
Deployed on Vercel: [vaultai-prod.vercel.app](https://vaultai-prod.vercel.app)
